🚗 TorqueX

TorqueX is a premium **3D sports car showcase** built using **HTML, CSS, and JavaScript** with **Three.js** and **GSAP** via CDN. The project delivers a cinematic, scroll-driven experience with interactive 3D controls and a modern luxury-inspired interface.

 ✨ Features

* Interactive 3D sports car
* 360° rotation and zoom
* Cinematic scroll animations
* Smooth camera transitions
* Responsive design
* Premium glassmorphism UI
* High-performance rendering
* Built with CDN libraries only

🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Three.js
* GSAP + ScrollTrigger

🚀 Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
3. No installation or build tools required.
